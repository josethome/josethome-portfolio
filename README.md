# :inbox_tray: Como baixar o projeto
```bash
    
    # Clonar o repositório
    $ git clone https://github.com/josethome/josethome-portfolio
    
    # Entrar no diretório
    $ cd josethome-portfolio

    # Instalar as dependências
    $ yarn install

    # Iniciar o projeto
    $ yarn start
    
```
